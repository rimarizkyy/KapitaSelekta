1. **LATAR BELAKANG MASALAH**

Perkembangan teknologi kini semakin maju dengan sangat pesat. Banyak teknologi teknologi bermunculan untuk mempermudah dalam penyelesaian masalah.  Para ahli dalam bidang pemrograman kini semakin dimudahkan dengan adanya layanan berbasis web seperti GitHub. Berikut adalah penjelasan segala hal yang berhubungan dengan GitHub.

1. **PEMBAHASAN**

1. **GitHub**

Sebuah web hosting yang melayani penyimpanan file - file pengembangan software atau perangkat lunak yang berbentuk proyek – proyek dengan pendaftaran akun gratis. GitHub termasuk situs source coding yang popular hingga saat ini.

1. **SSH Key**

Sebuah kunci penghubung antara aplikasi GitBash dengan web GitHub sendiri. Jadi saat SSH Key sudah berhasil dimasukkan maka tidak perlu lagi login dengan memasukkan username dan password pada saat kita clone, commit, push, dan pull.

1. **Repository**

Repository di dalam GitHub adalah tempat penyimpanan pada proyek proyek yang dimasukkan ke dalam GitHub, biasanya berisi file yang berisi source code.

1. **GitBash**

GitBash adalah aplikasi yang memudahkan untuk menunggah file source code yang biasanya berupa file markdown ke dalam GitHub sendiri
